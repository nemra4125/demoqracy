About this Project

demoQRacy is a sample project developed by Google's Developer Relations team. In addition to its voting feature, it demonstrates how you can build amazing web apps using the robust stack of Google developer products. In this project, we use the following technology to create a full fledged application:

Google Tech

  • App Engine
    • Python 2.7 Runtime
    • Account Authentication
    • Channel API
    • Task Queues
    • Mail Service
  • Charts API
  • Project Hosting
  • In-app Payments
  • Chrome Web Store
  • Chrome Frame
  • AdSense
  • WebFonts
  • +1 Button
  • Google Analytics

Third Party Tech

  • HTML5
    • Web Workers
    • Forms
  • jQuery
  • jQuery UI
  • jQuery UI Timepicker
  • jQuery Countdown
  • Jinja Templates

About Our Name

A "QR code" is a type of barcode that can contain any type of data: a phone number, a website URL, or even the text of War and Peace (if the code were large enough). At demoQRacy, we use QR codes to make democracy a little more high tech!
